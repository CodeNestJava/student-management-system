# Student Management System (Java)

A beginner-friendly Java project that manages student records.

## Features
- Add student
- View students
- Update student details
- Delete student

## Concepts Used
- Object-Oriented Programming (OOP)
- Arrays / Collections
- File handling (optional)

## Purpose
Built for learning and helping Java beginners understand real projects.

## How to Run
1. Open Main.java in your Java IDE
2. Compile and run
3. See the results in the console
