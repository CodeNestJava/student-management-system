# Student Management System (Java)

A console-based Java application to manage student records.  
This project is designed for beginners and demonstrates clean code, OOP concepts, and practical Java programming skills.

---

## Features
- Add new students (name & ID)
- View all students
- (Future) Update student details
- (Future) Delete student records

---

## Concepts Used
- Object-Oriented Programming (OOP)
- ArrayList Collections
- Encapsulation
- User input handling via Scanner
- Menu-driven console application

---

## Project Structure
```
student-management-system/
│
├── src/
│   ├── Main.java       # Main program with menu and logic
│   └── Student.java    # Student class representing each student
├── README.md           # Project description and instructions
```
---

## How to Run
1. Clone the repository:  
git clone https://github.com/CodeNestJava/student-management-system.git
2. Open the project in any Java IDE (Eclipse, IntelliJ, NetBeans)  
3. Compile and run `Main.java`  
4. Follow the console menu to add or view students
